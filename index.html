TECH SKILLS

프론트엔드 개발을 시작으로 3개월 동안 HTML, CSS, JavaScript를 집중적으로 학습하며 웹 개발의 기초를 다졌습니다. React를 사용하여 간단한 웹 애플리케이션을 만들었고, 백엔드 개발에서는 Node.js와 MySQL을 활용하여 기본적인 데이터베이스 연동을 경험했습니다. 또한, Spring Boot를 통해 서버 개발을 처음 접했으며, GitHub를 사용하여 소스 코드를 관리하고 버전 관리를 배웠습니다.

'당근 마차'라는 카풀의 팀프로젝트 와 개인 'OTT 프로젝트'를 통해 실무에 더욱 가까운 경험을 쌓았습니다. 이 과정에서 다음과 같은 기술적 역량을 크게 향상시켰습니다:

  • React를 활용한 동적이고 반응형 웹 애플리케이션 구현
  • Node.js와 Express.js를 이용한 백엔드 서버 구축 및 API 개발
  • MongoDB를 사용한 NoSQL 데이터베이스 설계 및 관리
  • YouTube Data API, TMDB API, TuneFind API, Vibe API 등 다양한 외부 API 통합 및 활용
  • Cloudtype을 이용한 프론트엔드와 백엔드의 통합 배포 경험
  • CORS 이슈 해결 및 보안 관련 설정 강화
  • Git을 활용한 버전 관리 및 협업 워크플로우 개선

프로젝트 진행 중 발생한 다양한 오류들(CORS 문제, API 호출 오류, 프론트엔드와 백엔드 간 통신 오류 등)을 해결하면서 문제 해결 능력을 크게 향상시켰습니다. 특히, API 통합 과정에서 겪은 어려움들을 극복하며 RESTful API에 대한 이해도를 높였고, 비동기 프로그래밍 스킬을 향상시켰습니다.

이 프로젝트를 통해 API와 서버의 중요성, 프론트엔드와 백엔드의 효과적인 통합, 그리고 Git을 활용한 효율적인 개발 프로세스의 중요성을 깊이 이해하게 되었습니다. 또한, 예상치 못한 기술적 문제들을 끊임없이 해결해 나가는 과정에서 개발자로서의 문제 해결 능력과 끈기를 기를 수 있었습니다.

현재도 최신 웹 개발 기술과 클라우드 서비스, 그리고 다양한 API 활용 방법을 지속적으로 학습하고 있습니다. 특히 서버리스 아키텍처, 마이크로서비스, 그리고 컨테이너화 기술에 관심을 가지고 학습 중입니다. 앞으로도 새로운 기술에 대한 도전을 두려워하지 않고, 끊임없는 학습과 실험을 통해 더욱 성장하는 개발자가 되고자 합니다.
